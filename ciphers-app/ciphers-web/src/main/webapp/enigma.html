About the Enigma Machine Cipher

How to Use This Page

You can use the Enigma Cipher page to enciper or decipher a message by following these steps:

  1. Set Enigma Parameters. Set the parameters of the Enigma Machine using the first form below. Here's how to set each parameter:
    • Plugboard Pairs: To set the plugboard pairs, choose 10 pairs of distinct letters (i.e., no letter can be included in more than one pair) to be swapped, and enter them in each of the 10 boxes.
      Example: to swap A and Q, write AQ or aq in the box.
    • Rotors: To set the rotors for the enigma machine, choose 3 unique rotors (i.e., no rotor can be used twice) in the order you want to use them.
      Example: to use rotor I on the left, V in the middle, and II on the right, set the rotor boxes to I V II. This step is order-sensitive: I V II is not the same as II I V. These will produce a different result.
    • Reflector: To set the reflector for the enigma machine, select which reflector to use. There is only one reflector, so there is no invalid input.
      Example: to use reflector B, set the reflector box to B.
    • Rotor Rings: To set the rotor ring settings, select the appropriate setting for the appropriate rotor position.
      Example: to use rotors I - IV - III with the left rotor (I) having a ring value of Q, the middle rotor (IV) having a ring value of N, and the right rotor (III) having a ring value of S, set the rings to Q N S.
    • Rotor Positions: To set the rotor position settings, select the appropriate setting for the appropriate rotor position.
      Example: to use rotors I - IV - III with the left rotor (I) having a starting position value of P, the middle rotor (IV) having a starting position value of E, and the right rotor (III) having a starting position value of T, set the starting positions to P E T.
    • Save Path: To set the save path for the enigma model, enter a fully qualified file path from the root drive, including any folders and the file name, with its extension.
      Examples: C:/enigma.bin, C:/Users/Owner/Documents/Ciphers/Models/enigma.bin
    Once all settings are configured, press the button labeled "Set Parameters" to initialize the Enigma model.
  2. Encipher a Message. Use the second form to encipher a message. Enter your message in the text box and press the "Encipher Text" button. This form can also be used to decipher text. Simply set up the model using the previous step, or import a model from a file using the third form.
    Note: The Enigma Machine must be initialized before it can encipher text.
  3. Reset the Enigma Machine. To send another message with the same starting parameters, or to import an existing model from a friend to decipher your message, enter a fully qualified file path from the root drive, including any folders and the file name, with its extension and select the "Reset Enigma" button.

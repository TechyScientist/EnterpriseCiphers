About the Caesar Cipher

The Caesar Cipher is a ciphering algorithm containing two "alphabets": one of which is the normal alphabet (A-Z in order). The other alphabet is the key to the cipher. For each letter in the message, it is replaced with the letter at the corresponding position in the key alphabet.


The Caesar Cipher algorithm is symmetric. In cryptography, an algorithm is defined as symmetric if the enciphered text (or ciphertext) can be given to the same algorithm as an input and the same plain text will be generated. We can say the Caesar cipher is symmetric because we know both alphabets when we encipher or decipher a message. The only difference between enciphering text and deciphering text in the Caesar cipher is which alphabet gets read from to create the generated text.

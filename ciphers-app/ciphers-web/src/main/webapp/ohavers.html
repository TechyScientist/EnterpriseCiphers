Ohaver's Algorithm

Author: Johnny Console

About Ohaver's Algorithm

This cipher is based on Kattis problem False Sense of Security (Kattis ID: falsesecurity), Summarized below:


  • The algorithm is based on modifying a morse code (dots and dashes, with words separated by spaces) string.
  • A morse code string is generated alongside a string of numbers, representing the length of each encoded character.
  • Spaces are removed from the morse string, so that without the key (the number string), the string is unreadable.
  • The number string is then reversed, and the morse string is decoded with those values, creating the ciphertext.
  • Due to the nature of the algorithm, using the ciphertext in the algorithm will return the plaintext string.
  • Note: This algorithm does not support using spaces, numbers or any other symbols not defined in the table below. Spaces in the input string will automatically be converted to the underscore "_" character.
